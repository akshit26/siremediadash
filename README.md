# Sire Media Landing Page

A Next.js single-page experience for Sire Media featuring the animated influencer discovery hero, stat counters, filter pills, and creator cards from the supplied mock.

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser to view the page.

## Scripts

- `npm run dev` – start the Next.js dev server.
- `npm run build` – create an optimized production build.
- `npm start` – run the production server after building.
- `npm run lint` – lint the project with Next.js defaults.
